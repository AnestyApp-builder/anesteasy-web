## 🚀 **Instalação e Uso**

### **Pré-requisitos**
- Node.js 18+
- npm ou yarn

### **1. Clone o repositório**
```bash
git clone https://github.com/seuusuario/anesteasy-web.git
cd anesteasy-web
```

### **2. Instale as dependências**
```bash
npm install
```

### **3. Execute o projeto**
```bash
# Desenvolvimento
npm run dev

# Build de produção
npm run build

# Executar build
npm run start

# Linting
npm run lint
```

### **4. Acesse a aplicação**
- **Desenvolvimento**: http://localhost:3000
- **Produção**: Após build, execute `npm run start`
