## 🚨 ALERTAS IMPORTANTES

### ❌ NÃO FAZER (Crítico)
1. **Não inventar depoimentos** - É fraude e clientes descobrem
2. **Não usar números falsos** - "10.000 usuários" sem ter
3. **Não ativar seção de depoimentos sem dados reais**
4. **Não exagerar resultados** - Ser conservador é mais confiável

### ✅ FAZER (Recomendado)
1. **Começar conservador** - "Mais de 50 anestesiologistas" é ok
2. **Ser transparente** - "App mobile em breve" gera confiança
3. **Mostrar o real** - Screenshots reais > mockups bonitos
4. **Iterar rápido** - Teste pequeno → Medir → Ajustar → Repetir
